GLB to 3D Tiles Helper

Cesium ion 非依存で、OSM ベースマップ上に地理配置つき 3D Tiles の設定確認と GLB 仮配置を行うための支援ページです。 軽量代理表示モードと、必要時だけ本体 GLB を出す手動表示モードを切り替えられます。

概要

このページは、環境構築の確認 と 変換条件の整理・JSON雛形生成 を目的にした GitHub Pages 向けUIです。 地図表示は CesiumJS + OSM を前提にし、Cesium ion 依存を避けています。

前提条件

・Node.js / npm がインストール済みであること
・必要に応じて Python 仮想環境が利用できること
・作業フォルダに変換対象の .glb があること
・作業フォルダで 3d-tiles-tools が使えること
・ローカル確認時は file:// 直開きより http://localhost 推奨

セットアップ手順

cd /your/project/path

python3 -m venv .venv
source .venv/bin/activate

python -m pip install --upgrade pip
python -m pip install trimesh numpy pygltflib

npm install 3d-tiles-tools
npx 3d-tiles-tools --help

python3 -m http.server 8000

3. 表示モード

軽量代理表示は向き合わせ用の常時表示です。GLB は重い場合があるため、必要時だけ手動表示にしています。

4. 分割条件

モデルをどの空間単位で分割するかを選びます。
Quadtree は XY 平面の4分割、Octree は XYZ の8分割です。
子タイル表示時に親タイルをどう扱うかを示します。
ADD は親子を重ねる考え方、REPLACE は子が出ると親を置き換える考え方です。
1つの末端タイルに含める三角形数の上限目安です。
分割を何段まで再帰的に続けるかの上限です。
1タイルあたりのサイズ目安です。将来の再分割判断に使います。
