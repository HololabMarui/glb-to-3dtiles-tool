GLB to 3D Tiles Helper

左で変換条件、右で地図作業と出力確認を行う構成です。設定JSONを保存し、ローカルのPythonスクリプトで変換を実行します。

環境構築ガイド

準備するもの

・Python 3.x
・このページで保存した stage2_config.json
・変換スクリプト build_stage2_tiles.py
・依存パッケージ定義 requirements.txt
・変換対象の .glb ファイル

Mac / Linux

python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
python build_stage2_tiles.py stage2_config.json

Windows

py -3 -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python build_stage2_tiles.py stage2_config.json

実行前の確認

・stage2_config.json の sourceGlb が実在するローカルパスになっていること
・output.outDir が書き込み可能なフォルダであること
・output.outputFormat が glb または b3dm になっていること

requirements.txt の例

trimesh
numpy
pygltflib

出力されるもの

・tileset.json
・分割されたタイル群（.glb または .b3dm）

変換設定

地図上で位置合わせと GLB の仮配置表示ができます。設定を保存したら、ローカルの Python スクリプトで変換してください。

入力GLB

出力方式

GLB content は新しめの構成、B3DM content は既存互換向けです。
