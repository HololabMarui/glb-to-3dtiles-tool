GLB → 3D Tiles 変換ツール

GLBファイルをブラウザ上でメッシュ分割し、地理参照付き tileset.json と chunk GLB を ZIP で出力します。
位置・向きはすべて手動調整値です。自動推定は行いません。

このツールについて

GLBファイルをブラウザ上でメッシュ単位に分割し、root.transform 付きの tileset.json を生成する Web ツールです。
サーバー不要・GitHub Pages でそのまま動作します。 B3DM 変換は対象外です（Node.js 側で別途実行してください）。

前提条件

  • モダンブラウザ（Chrome / Edge / Firefox 最新版）
  • GLBファイルが手元にあること
  • 配置したい緯度・経度・高さを事前に把握していること
  • B3DM化が必要な場合は Node.js + npx 3d-tiles-tools を別途用意

使い方

  • 1. GLB を選択 — ファイルを選ぶかドロップする
  • 2. チャンク数を設定 — メッシュ数以下を目安に設定
  • 3. 地図で位置を調整 — マーカーをドラッグ、またはフォームに直接入力
  • 4. Heading / Pitch / Roll を入力 — 向きを手動で指定
  • 5. 「変換 & ZIP ダウンロード」ボタンを押す
  • ZIP に tiles_src/*.glb と tileset.json が含まれます

出力ファイル構成

output.zip
  tiles_src/
    chunk_000.glb
    chunk_001.glb
    ...
  tileset.json
  build_report.json

注意事項

  • 位置・向きはすべて 手動調整値 です。自動推定は行いません。
  • チャンク分割はラウンドロビン（Python版と同一アルゴリズム）です。
  • B3DM 化はこのツールでは行いません。必要な場合は npx 3d-tiles-tools glbToB3dm を使ってください。
  • 大きなGLBファイルはブラウザのメモリを消費します。

用語

  • チャンク: GLBを分割した部分ファイル（chunk_000.glb など）
  • tileset.json: 3D Tiles のエントリーポイント。root.transform に地理座標系への変換行列が入ります。
  • ECEF: 地球中心・地球固定座標系。3D Tiles の transform はこの座標系で指定します。
  • ENU: 東・北・上 の局所座標系。ECEF との変換に使います。
  • Heading: 北を基準とした回転角（Z軸回転）
  • Pitch: 前傾き（Y軸回転）
  • Roll: 横傾き（X軸回転）
