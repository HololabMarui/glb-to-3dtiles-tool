GLB to 3D Tiles Helper

環境構築ガイドと、GLB→B3DM / tileset.json 生成支援ツールを分けて使える GitHub Pages 向けページです。

概要

このツールは、3d-tiles-tools を使って GLB → B3DM と tileset.json 生成を支援するためのものです。GitHub Pages 上では npm コマンドの実行自体は行わず、手順確認・入力補助・コマンド生成 を担当します。

前提条件

・Node.js / npm がインストール済みであること
・作業フォルダに変換対象の .glb があること
・作業フォルダで 3d-tiles-tools が使えること

セットアップ手順

npm install 3d-tiles-tools
npx 3d-tiles-tools --help

上記コマンドが通れば、変換ツールページで生成したコマンドをそのまま実行できます。

基本手順

1. Node.js / npm をインストール
2. 作業フォルダで npm install 3d-tiles-tools を実行
3. 変換ツールページで入力値を設定してコマンドを生成
4. glbToB3dm で B3DM 作成
5. updateAlignment で alignment 補正
6. createTilesetJson で位置つき tileset を作成
7. torinome では tileset.json を読み込む

注意事項

・緯度・経度・高さ・向き（Heading / Pitch / Roll） は、このページが自動計算するものではありません。
・これらの値は、現地確認、既存GIS、ビューア上の目視調整、外部情報などをもとに 手動で設定・調整 してください。
・特に高さと向きは、モデル原点や地形条件によってズレやすいため、torinome 上で確認しながら再調整する前提です。

入力

このページは 手動で決めた位置・向き・高さ をもとにコマンドと tileset.json を生成します。GitHub Pages 上では npm 実行はできません。
