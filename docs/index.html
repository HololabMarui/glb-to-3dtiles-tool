GLB Split to 3D Tiles Helper

GLB を複数チャンクへ分割し、root.transform 付きの tileset.json を生成するヘルパーです。content.uri を glb / b3dm で切り替えできます。

概要

このページ自体は変換処理をしません。ローカルの glb_tiles_georef.py を実行するための入力補助です。tileset.json の content.uri を aligned.b3dm にするか chunk_000.glb にするかを切り替えられます。

前提

  • glb 出力だけなら numpy と trimesh
  • b3dm 出力には npx 3d-tiles-tools
  • Python と HTML を同じフォルダに置くとダウンロード共有しやすいです

動作イメージ

GLB 1個
↓
メッシュ単位で 2〜3 チャンクに分割
↓
chunk_000.glb / chunk_001.glb / ...
↓
content-format=glb の場合:
  tileset.json から tiles_src/*.glb を参照

content-format=b3dm の場合:
  glbToB3dm → updateAlignment
  tileset.json から tiles/*.b3dm を参照

入力

Pythonをダウンロード

glb 参照に切り替える場合は --content-format glb を使います。skip-b3dm は glb モードと組み合わせるのが自然です。
