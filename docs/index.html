GLB Split to 3D Tiles Helper

GLB を 2〜3 チャンク程度の複数 B3DM に分割し、root.transform 付きの tileset.json を生成する位置反映版 Python スクリプト向けヘルパーです。

概要

このページ自体は GitHub Pages 上で変換処理をしません。ローカルの glb_tiles_georef.py を実行するための入力補助・コマンド生成 を行います。ページ内の Pythonをダウンロード ボタンから同じフォルダのスクリプトも配布できます。

前提

  • Python 仮想環境があること
  • trimesh が入っていること
  • npx 3d-tiles-tools --help が通ること
  • glb_tiles_georef.py をこのHTMLと同じ場所に置くと、ページ内のダウンロードボタンでも共有できます

動作イメージ

GLB 1個
↓
メッシュ単位で 2〜3 チャンクに分割
↓
chunk_000.glb / chunk_001.glb / ...
↓
glbToB3dm → updateAlignment
↓
chunk_000_aligned.b3dm / ...
↓
root.transform 付き tileset.json

注意

  • この lite 版は メッシュ単位 の簡易分割です。単一メッシュの GLB は分かれにくいです。
  • 緯度経度は root.transform に書き込みますが、高さや向きはモデル原点の影響を受けます。
  • まずは heading / pitch / roll = 0 で位置だけ確認するのが安全です。

入力

Pythonをダウンロード

出力される tileset.json は root.transform を持つ前提です。表示位置がずれる場合は、モデル原点や高さの調整が必要です。
